POI Dataset
- No : data number
- id : id of POI
- name : name of POI
- type : type of POI
- lattitude : lattitude
- longitude : longitude
- tarif : entrance fee of POI
- duration : avg. duration of tourist spend in POI
- rating : rating of POI


POI Time Matrix
- id : id of POI
- id_a,id_b : index -> Time Matrix[id_a,id_b]
- durasi : time duration between POI id_a to POI id_b
